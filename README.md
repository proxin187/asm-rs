# asm-rs

> [!WARNING]
> This assembler is a hobby project and shall not be used for anything professional

Asm-rs is an assembler for x86_64 linux with intel syntax.

## Features
 - Labels
 - Instructions supported
   - Mov
   - Cmp
   - Jcc (jmp, je, jg, jb)
   - Binary expr (add)
   - Stack (push, pop)
